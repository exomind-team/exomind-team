## 协作规范

- 个人内容请勿提交到此仓库
- 使用 `[[wikilink]]` 在 Obsidian 中跨库引用
- 定期同步更新子模块：`git submodule update --init --recursive`
- 提交前检查子库状态

## 与个人库的关系

本仓库作为个人 Obsidian 保险库的子模块存在：

```
个人库/
└── ExoMind-Team/   ← 指向本仓库
```

### 同步流程

1. 团队库更新后推送到远程
2. 个人库执行 `git submodule update` 拉取最新
3. 或直接 `cd ExoMind-Team && git pull`
