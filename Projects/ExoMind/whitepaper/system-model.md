# 真相模型与数据契约

## 目标

- 明确"以事件为事实"的系统不变量
- 给推断边界设定可审计规则

## 唯一事实：EventLog = 日记

- EventLog 事件是系统唯一事实
- **日记是 EventLog 的实践**——按时间顺序记录自己的事实，不可修改
- 仅追加，不 UPDATE、不 DELETE
- 所有派生（报告、视图、统计）都来自事件

## 日记实践

- **位置**：`2-个人状态与历史记录/日记/`
- **格式**：按日期命名的 Markdown 文件
- **原则**：记录发生的事实，不删除、不修改历史

## 推断边界

- 推断只能产出派生结果，不得写回 EventLog/日记
- 推断必须可解释、可撤回、可被覆盖
- 推断不替代人类决策
- 人类意志优先于系统推断
