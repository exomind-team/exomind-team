# 外心白皮书索引

本目录是白皮书的概述，目标是把共识收敛为可沟通、可执行、可审计的文本。

## 一句话定位

外心是一套"**本地优先**、**以事件为事实**、**人类主导**、**可解释**"的**个人成长系统**。

## 核心不变原则

- **EventLog** 是唯一事实（日记是其实践——按时间顺序记录自己的事实，不可修改）
- 动态知识管理是次要功能
- Human agency（人类主导）优先于系统推断

## 主干流程

```
外部缓冲区 → 身心状态觉察 → 任务管理 → 执行 → 反馈
   (1)           (2)          (3)      (4)    (5)
```

当前阶段：**外部缓冲区**（Scout 侦察兵）

## 术语表

- ExoMind（外心）：项目名称，个人成长系统

- Human agency（人类主导）：人类意志优先于系统推断
