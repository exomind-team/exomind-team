# 术语表（Glossary）

## 核心术语

- **ExoMind（外心）**：项目名称，个人成长系统
- **EventLog（事件日志）**：唯一真相层，append-only。**日记是其实践**——按时间顺序记录自己的事实，不可修改
- **Human agency（人类主导）**：人类意志优先于系统推断
- **Local-first（本地优先）**：默认本地存储与控制权

## 主干流程术语

- **ExoBuffer（外部缓冲区）**：Scout 侦察兵前哨站，主干流程第1阶段
- **Scout（侦察兵）**：信息价值评估 Agent
- **Session / TimeBlock**：可启动的行动闭环，主干流程第4阶段

## 架构术语

- **Truth Layer（事实层）**：EventLog = 日记
- **Surface Layer（入口界面层）**：多入口（Obsidian、浏览器、手机等）统一到单一真相
