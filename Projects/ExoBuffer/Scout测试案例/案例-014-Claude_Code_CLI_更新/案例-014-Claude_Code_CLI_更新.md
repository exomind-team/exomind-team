## 1. 正文内容

一句话总结：更快、更稳、更像"本地 AI 编程搭子"了。

✨ 这次更新最值得看的点：

🔥 **技能体验大升级**
- Skills 支持自动热更新，改完立刻生效
- Skills / Slash Command / Agent 全面支持 Hook
- 支持 context: fork，子代理并行更顺畅

🌍 **多语言 & 可控性更强**
- 新增 language 配置（可直接指定日语 / 中文等）
- 支持禁用指定 Agent / 限制可用工具
- Gitignore 可控，项目级文件选择更干净

⌨️ **CLI & 编辑体验狂飙**
- Vim 操作补全一大波（y / p / 文本对象 / 缩进...）
- Shift+Enter、Ctrl+B 后后化统一支持
- Slash 命令随处补全，效率拉满

🛡️ **稳定性 & 安全感拉满**
- 修复敏感信息泄露风险（Token / Key）
- 大量并发、恢复、后台任务相关 bug 修复
