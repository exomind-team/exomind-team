> <span style="color: rgb(255, 0, 0); font-weight: bold">美国侵略委内瑞拉，非法逮捕马杜罗，是近年来中美博弈中，我们丢的第一分，也是很重的一分。</span>
>
> <span style="color: rgb(255, 0, 0); font-weight: bold">但如果认真总结反思，利用好这次事件，我们也将获得史无前例的机遇。</span>

我会在后面陆续文章中，陆续分析为什么当下是失分，为什么不应事不关己，而未来，为什么是机遇。

但在此之前，今天，先说最重要的一件事。

我在之前《这是中国最后一次机会》（已阵亡，知乎还有）中提过一条政策建议：

<span style="font-weight: bold">开展两年打击包括日美台在内的内奸专项行动。</span>

为什么？

看最近的伊朗和委内瑞拉就知道。

委内瑞拉昨天的应对，错失了最佳时机，，回头看，根本不应如此。

<span style="font-weight: bold">1.美军直升机进入的时候，没有任何抵抗。</span>

轰炸持续了一个小时，看似是因为打击瘫痪的，但，根本不是。

一方面，是马杜罗自己的侥幸。

一会儿斗争，一会儿服软，放话让特朗普来投资，自己本身就根本没真下武力斗争的决心。

不论是军队，还是马杜罗，都没做好准备。

连舔也门胡塞武装脚底板的资格都没有。

人家围了那么久，自己还在ZZ表演。

以战方能止战，如果早早就进入战争状态，美国评估损失很大，自然就会三思。

如此暧昧，很容易被人看穿。

另一方面，是战争发生时的松懈。

按常理说，在重兵围困，对方情报人员一直在境内活动的情况下，但凡有风吹草动，马上就应该全民皆兵。

事实上，从上到下，睡得死猪一样。

而一些士兵，在最开始听到爆炸后就丢弃阵地，跑路了，连最基本的抵抗都没有，所谓几百万民兵，外强中干。

这与伊拉克当时一模一样。

委内瑞拉有大量俄罗斯援助的单兵防空导弹，如果民兵真的管用，这几架飞机根本飞不过去。

更不要说，地面防空火力压根基本就没开火。

一旦有一点受阻，整个形势会完全不同。

<span style="font-weight: bold">2.马杜罗是被内奸卖的</span>

CIA早就下场了，马杜罗夫妇是在睡梦中被拖走的。

他们有个地堡，马杜罗都没来得及进去就被抓了，身边最亲的人必然提供消息。

生活作息、位置信息、基本情况一清二楚。

连总统府的卫队，如果组织有效反击，美军那么点人，根本没办法，再发动第二波，就跟俄罗斯结果差不多。

说明，从防空部队，到指挥所，到身边人，相当部分被买通了。

内部跟筛子一样。

但是，第一枪打响后，什么反应都没有。

委内瑞拉内部，已经被美国严重渗透。

关键BM，关键岗位，关键人物，大概都被收买。

与伊拉克，伊朗，利比亚，何其相似。

这种情况下，我们还指望什么后续人民群众的汪洋大海，个人觉得，内部争夺利益内乱的可能性更大。

<span style="color: rgb(255, 0, 0); font-weight: bold">接下去，特朗普的动作远不如此，他的做法会震惊世界，但这，也必然刺激他膨胀，导致像普京顺利拿到克里米亚之后的挫折。</span>

我们怎么做，怎么反应，如何谋略，很重要。

因为，已经错过了一次极好把美国拉下泥潭的机会。

前天的会面，本就是战略误判，以为能吓唬得了谁，想多了。我们一定要加强战略研判，而非情况分析，不然，只能是一次次“大吃一惊”。

这都后续再说，今天最重要的提醒是：

<span style="color: rgb(255, 0, 0); font-weight: bold">堡垒，都是从内部开始攻破的。</span>

<span style="color: rgb(255, 0, 0); font-weight: bold">武器、装备，都只是纸面数据，不是最核心的东西，敢不敢用，人心向谁，意志如何，关键时刻见分晓。</span>

在我们周围，也多的是，不再只是要盯着反F了，人心和意识形态的甄别、评价、筛选，更为重要。
