# 02-集体劳动提交

## 职责

- 从个人日记中收集劳动记录
- 剥离隐私信息，汇总到集体劳动日志
- 自动化 Git 提交流程
- 管理评审流程

## 核心能力

- 📖 识别和提取劳动记录
- 🔒 隐私信息剥离
- 🔀 Git 分支创建和提交
- 📝 PR 模板生成
- 📊 劳动统计汇总

## 使用方式

```bash
# 启动 Agent
请整理今天的集体劳动记录并提交 PR。
```
