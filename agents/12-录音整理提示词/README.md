# 录音整理提示词（Prompt Pack）

本文件夹沉淀“把录音转写原文整理成可读日记”的提示词模板（prompts/提示词），与仓库内的 `diary-manager（日记管理）` 工作流兼容。

## 目标产物（Outputs）

1) **录音原文单行版**：每条发言一行、无空行，统一为  
`发言人，时间戳，内容`

2) **可折叠录音块（Obsidian callout）**：将整段录音放入  
`> [!details]- 录音原文（可折叠）`

3) **摘要/总结/分析/待办（Obsidian callout）**：将会议内容进行“科学家式分析（scientific analysis/科学分析）”，并从原文抽取待办（todo list/待办清单，必须引用原话）。

## 文件清单

- `output/12-录音整理提示词/01-录音转写整理-单行格式提示词.md`
- `output/12-录音整理提示词/02-Obsidian-Callout-封装提示词.md`
- `output/12-录音整理提示词/03-会议摘要-科学家式分析提示词.md`
- `output/12-录音整理提示词/04-实现评审与改进建议.md`（评审记录，用于提示词迭代）

## 使用方式（推荐顺序）

1. 先用 `01-...` 把原始转写整理成单行格式（无空行）。
2. 再用 `02-...` 把单行版录音整体放进可折叠 callout（提示块）。
3. 最后用 `03-...` 生成摘要/总结/分析/待办 callout，并补齐关键名词的中英对照（如：`CPU（Central Processing Unit/中央处理器）`）。
